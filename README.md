# 🏥 Hospital Management System (MERN Stack)

A full-stack Hospital Management System built using the MERN stack to efficiently manage patients, doctors, departments, and appointment scheduling in a healthcare environment.

---

## 🚀 Features

- 🔐 Secure authentication & authorization (JWT-based)
- 👨‍⚕️ Role-based access (Admin / Doctor / Staff / Patient)
- 📅 Appointment booking & management system
- 🧑‍🤝‍🧑 Patient & doctor management
- 🏥 Department management
- 📊 Admin dashboard
- 🌐 Responsive UI using React.js & Tailwind CSS
- ⚡ RESTful API architecture
- 🗄️ Scalable MongoDB database design

---

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/your-username/hospital-management-system.git
cd hospital-management-system
2. Install dependencies
# Backend
cd server
npm install

# Frontend
cd ../client
npm install

3. Environment Variables

Create a .env file inside the server folder:
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key


4. Run the application
# Start backend
cd server
npm run dev

# Start frontend
cd client
npm start
